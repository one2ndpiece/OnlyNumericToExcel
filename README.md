# 学習用
色々自動化していた中で結構汎用性もあって便利なスクリプト
まだファイルとして使いやすい形にはしていない
# 用途
テキストファイルの数字だけを行ごとに抽出するもの
次のテキストファイルが
```txt 
あ123い-456う
え132おおおおおお6さｄふぁｓ0あｓｄ
```

以下のように変換される。
| A   | B    | C   |
| --- | ---- | --- |
| 123 | -456 |     |
| 132 | 6    | 0   |

意外と便利だった・・・
